# 간단한 계산기 (Simple Calculator)
<br>
✨ 기능 <br>
✅ 사칙연산 지원 (+, -, *, /) <br>
✅ 기록 저장 <br>
✅ EXIT 입력 시 종료 가능 <br>
<br>
🚀 사용 방법 <br>
visual Studio code , intellij IDEA 코드복사 후 실행 <br>
